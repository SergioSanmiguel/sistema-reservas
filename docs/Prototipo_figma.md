## 2. Descripción General
Prototipo visual para la interfaz web y móvil del sistema de reservas.


### Objetivos del prototipo
- Visualizar el flujo de usuario completo: registro → login → reservas → calendario → notificaciones.
- Establecer una guía de estilos (colores, tipografía, espaciado, botones, etc.).


## 3. Principales Pantallas
| Pantalla | Descripción |
|-----------|--------------|
| Login / Registro | Permite autenticación de usuarios. |
| Calendario | Muestra disponibilidad de espacios y reservas activas. |
| Crear Reserva | Formulario para seleccionar espacio y horario. |
| Panel Admin | Gestión de espacios, usuarios y reservas. |
| Notificaciones | Vista con avisos de cambios de estado. |
